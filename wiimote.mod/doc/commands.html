axe.WiiMote: Functions Modinfo Source  

Wiimote Driver for Windows and MacOS 10.4+

Talk to Wiimotes with a bluetooth enabled PC.

Functions Summary

CloseWiimotes Disconnects all Wiimotes.
OpenWiimotes Listen for Wiimotes.
PollWiimotes Polls all connected Wiimotes.
SetWiimoteLEDS Set status of Wiimote LED display.
SetWiimoteRumble Set status of Wiimote rumble motor.
WiimoteAxis Return axis values.
WiimoteButton Return button bits.

Function CloseWiimotes()
DescriptionDisconnects all Wiimotes.

Function OpenWiimotes()
DescriptionListen for Wiimotes.
InformationMay in future reset the library. Returns 0 if successful.
Example
Import axe.Wiimote

Print "OpenWiimotes()="+OpenWiimotes()

Local count,i,j

While True
	count=PollWiimotes()
	Print "count="+count
	For i=0 Until count*8
		Print "~tbutton["+i+"]="+WiimoteButton(i)
	Next
	For i=0 Until count*32
		Print "~taxiis["+i+"]="+WiimoteAxis(i)
	Next
	Delay 100
Wend

Function PollWiimotes()
DescriptionPolls all connected Wiimotes.
InformationReturns current number of connected Wiimotes, call this function to capture the state of all Wiimotes before using the WiimoteButton and WiimoteAxis commands.
